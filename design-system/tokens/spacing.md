# Spacing

## Grid

| Token | Value |
| --- | --- |
| `0.5` | `2px` |
| `1` | `4px` |
| `2` | `8px` |
| `3` | `12px` |
| `4` | `16px` |
| `5` | `20px` |
| `6` | `24px` |
| `8` | `32px` |

## Usage

- In the panel, `2` and `3` are the default inter-component gaps.
- Do not exceed `4` between siblings inside the 240x400 frame.
- Minimum hit target is `24px`.
