# Typography

## Scale

| Token | Size / Line Height | Usage |
| --- | --- | --- |
| `xs` | `11px / 14px` | Timestamp, helper labels |
| `sm` | `12px / 16px` | Default body copy in the panel |
| `md` | `13px / 18px` | Secondary headings |
| `lg` | `15px / 20px` | Project names, dialog titles |
| `xl` | `18px / 24px` | Daily report heading |

## Weight

- `regular` = 400
- `medium` = 500
- `semibold` = 600

No light weight is allowed. Small UI text depends on contrast and stable letterforms more than elegance.
